AngularJS within a style element not working
<p>I'm trying to use angular js bindings within a style element yet it doesn't seem to work.  Is this a bug and is there a better way to do this?</p>&#xA;&#xA;<pre class="lang-html prettyprint-override"><code>&lt;style ng-repeat="test in styles"&gt;&#xA;.test {&#xA;    background-color: {{ test.backgroundColor }};&#xA;    display: {{ test.display }};&#xA;    width: {{ test.width }};&#xA;    height: {{ test.height }};&#xA;    }&#xA;&lt;/style&gt;&#xA;</code></pre>&#xA;&#xA;<p><a href="http://jsfiddle.net/cmsanche/PpyVn/" rel="nofollow">http://jsfiddle.net/cmsanche/PpyVn/</a></p>&#xA;
<p>I don't think Angular parses style elements. You probably want to use the ngStyle directive:</p>&#xA;&#xA;<p><a href="http://docs.angularjs.org/api/angular.module.ng.$compileProvider.directive.ngStyle" rel="noreferrer">http://docs.angularjs.org/api/angular.module.ng.$compileProvider.directive.ngStyle</a></p>&#xA;&#xA;<pre><code>&lt;!doctype html&gt;&#xA;&lt;html ng-app&gt;&#xA;&lt;head&gt;&#xA;&lt;script src="http://code.angularjs.org/angular-1.0.0rc10.min.js"&gt;&lt;/script&gt;&#xA;&lt;/head&gt;&#xA;&lt;body&gt;&#xA;&lt;div ng-init="styles=[{backgroundColor: 'red', width:'100px', height: '100px', display: 'block'}]"&gt;&#xA;&#xA;&lt;div ng-repeat="test in styles" ng-style="test"&gt;&#xA;.test {&#xA;    background-color: {{ test.backgroundColor }};&#xA;    display: {{ test.display }};&#xA;    width: {{ test.width }};&#xA;    height: {{ test.height }};&#xA;    }&#xA;&lt;/div&gt;&#xA;&lt;/html&gt;&#xA;</code></pre>&#xA;&#xA;<p>Plunker (like jsFiddle, but more Angular-friendly) <a href="http://plnkr.co/edit/n70DQw?p=preview" rel="noreferrer"><strong>Demo</strong></a></p>&#xA;
<p>This now works in newer versions of Angular.</p>&#xA;&#xA;<p>Here <a href="http://jsfiddle.net/PpyVn/4/" rel="nofollow">http://jsfiddle.net/PpyVn/4/</a> is the OPs fiddle with 1.0 replaced with 1.3.0-rc.3.</p>&#xA;&#xA;<pre><code>&lt;script src="//ajax.googleapis.com/ajax/libs/angularjs/1.3.0-rc.3/angular.min.js"&gt;&lt;/script&gt;&#xA;</code></pre>&#xA;
