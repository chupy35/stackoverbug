Get several documents by alternative ID in ElasticSearch
<p>My documents have an alternative Id fields. I need to get them using it.</p>&#xA;&#xA;<p>So, I know I'm able to use a <code>multiget</code> operation, however, this one needs I provide the <code>ElsticSearch document id</code>.</p>&#xA;&#xA;<p>I'm actually trying something like this:</p>&#xA;&#xA;<pre><code>curl -XGET 'http://ESNode01:9201/living_team/inputs/_search?pretty' -d '{&#xA;  "query":{&#xA;     "filtered":{&#xA;        "filter":{&#xA;           "term":{&#xA;              "resources.source.sourceId.batch":"3fcb8905-a307-11e5-88de-382c4ab9e433"&#xA;           }&#xA;        }&#xA;     }&#xA;  }&#xA;}'&#xA;</code></pre>&#xA;&#xA;<p>How could I request a query in order for ElasticSearch to provide every document that has <code>resources.source.sourceId.batch eq to [id1, or id2, or ...]</code></p>&#xA;
