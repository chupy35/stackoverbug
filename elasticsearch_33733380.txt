How to query on multiple fields in elasticsearch?
<p>i have tried the multiple field query and it works fine. But I would like to know what other options are generally used to query multiple fields in elasticsearch?</p>&#xA;
<p>Structured queries with multiple terms, for finding exact values, the same as SQL</p>&#xA;&#xA;<p><a href="https://www.elastic.co/guide/en/elasticsearch/guide/current/_finding_multiple_exact_values.html" rel="nofollow">https://www.elastic.co/guide/en/elasticsearch/guide/current/_finding_multiple_exact_values.html</a></p>&#xA;&#xA;<pre><code>"bool" : {&#xA;                    "must" : [&#xA;                        { "term" : { "tags" : "search" } }, &#xA;                        { "term" : { "tag_count" : 1 } } &#xA;                    ]&#xA;                }&#xA;</code></pre>&#xA;
<p>For example, consider following sql query, </p>&#xA;&#xA;<pre><code>SELECT product&#xA;FROM   products&#xA;WHERE  (price = 20 OR productID = "XHDK-A-1293-#fJ3")&#xA;  AND  (price != 30)&#xA;</code></pre>&#xA;&#xA;<p>In these situations, you will need the bool filter. This is a compound filter that accepts other filters as arguments, combining them in various Boolean combinations.</p>&#xA;&#xA;<p>The Query DSL would be,</p>&#xA;&#xA;<pre><code>GET /my_store/products/_search&#xA;{&#xA;   "query" : {&#xA;      "filtered" : { &#xA;         "filter" : {&#xA;            "bool" : {&#xA;              "should" : [&#xA;                 { "term" : {"price" : 20}}, &#xA;                 { "term" : {"productID" : "XHDK-A-1293-#fJ3"}} &#xA;              ],&#xA;              "must_not" : {&#xA;                 "term" : {"price" : 30} &#xA;              }&#xA;           }&#xA;         }&#xA;      }&#xA;   }&#xA;}&#xA;</code></pre>&#xA;&#xA;<p>Follow the below link for documentation&#xA;<a href="https://www.elastic.co/guide/en/elasticsearch/guide/current/combining-filters.html" rel="nofollow">https://www.elastic.co/guide/en/elasticsearch/guide/current/combining-filters.html</a></p>&#xA;
