how to use phrase match in elastic.js?
<p>i want to use phrase match in elastic.js. &#xA;the docs says</p>&#xA;&#xA;<pre><code>"match": {&#xA;    "title": {&#xA;        "query": "quick brown fox",&#xA;        "type":  "phrase"&#xA;    }&#xA;}&#xA;</code></pre>&#xA;&#xA;<p>i want to do this</p>&#xA;&#xA;<pre><code>ejs.BoolQuery().must(&#xA;                    ejs.FilteredQuery(&#xA;                        ejs.MatchAllQuery(),&#xA;                        ejs.TermsFilter('expressNumber', ['as1243432','sd4324324']).type('phrase')&#xA;                    )&#xA;                );&#xA;</code></pre>&#xA;&#xA;<p>There is not enough examples.Any help?</p>&#xA;
<p>If you want to create a <code>phrase match</code> you need to use the <a href="http://docs.fullscale.co/elasticjs/ejs.MatchQuery.html" rel="nofollow"><code>MatchQuery</code></a>, not the <code>TermsFilter</code>. In your case, it should go like this:</p>&#xA;&#xA;<pre><code>ejs.BoolQuery().must(&#xA;    ejs.MatchQuery("expressNumber", "as1243432 sd4324324").type("phrase")&#xA;);&#xA;</code></pre>&#xA;
