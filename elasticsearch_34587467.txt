<p>You can use wildcards in the index name. If you want to search all documents in indices whose name start with "logs", the corresponding query is:</p>&#xA;&#xA;<pre><code>POST logs*/_search&#xA;{&#xA;    "query": {&#xA;        "match_all": {}&#xA;    }&#xA;}&#xA;</code></pre>&#xA;&#xA;<p>Read <a href="https://www.elastic.co/guide/en/elasticsearch/reference/current/multi-index.html" rel="nofollow">this</a> for more information.</p>&#xA;
