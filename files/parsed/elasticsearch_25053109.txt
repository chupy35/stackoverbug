It works but I need to search based on Id of document.

    
    
      SearchResponse response = client.prepareSearch("dbIndex")
        .setSearchType(SearchType.QUERY_AND_FETCH)
        .setQuery(QueryBuilders.termQuery("_id","532adfdxfe8343000458ffd3"))
        .setExplain(true)
        .setQuery().setExplain(true)

Use `prepareGet` and set the ID.

    
    
    GetResponse response = esClient.prepareGet()
        .setIndex("dbIndex")
        .setType("yourType")// set the index type as well
        .setId("532adfdxfe8343000458ffd3")
        .setFields(new String[]{"aa.ct", "aa.bb", "a.aa"})
        .execute()
        .actionGet();
    

You can use get method to search for an _id. (Its not actual searching but a
kind of that) It returns the record if it really exist, else in the json
response, it will say, exists: false.
