I had the same issue: I installed `React Developer Tools` plugin, but the
`React` tab didn't appear. The solution was simple: I just closed the Chrome
tab and opened it again - and it was there.

You can read about other possible reasons on their official GitHub page
<https://github.com/facebook/react-devtools#user-content-the-react-tab-doesnt-
show-up>

This was happening to me because I didn not have global React object, or a
global require that is able to require('react') from global scope.  
I was able to make the extension work by requiring react in my gulpfile. I'm
sure it would work the same in a webpack file:

    
    
    var React = require('react');
