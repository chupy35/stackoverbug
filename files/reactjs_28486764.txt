how to call function in parent component
<p>There's a fiddle for this question <a href="https://jsfiddle.net/4TpnG/504/" rel="nofollow">here</a>I have a FormControl component in a React.js app in which I render several different types of other components. For example, in the code below, you can see that I render a CheckBoxWithLabel. You will notice that I pass an a function </p>&#xA;&#xA;<pre><code>onCheck={this.toggleCheckBox}&#xA;</code></pre>&#xA;&#xA;<p>so that in the CheckBoxWithLabel component (code not shown) I can call it with the onChange event (a built in event for checkboxes in react)</p>&#xA;&#xA;<pre><code>onChange={this.props.onCheck} //this code's in the CheckBoxWithLabel&#xA;</code></pre>&#xA;&#xA;<p>So the result is that when the checkbox is clicked, the <code>toggleCheckBox</code> function in the parent component is called, but this is only working for the first <code>CheckBoxWithLabel</code> shown below, the others (which are dynamically created in the second return function) are not able to be checked, and hence the toggleCheckBox function is not getting called.</p>&#xA;&#xA;<p>Why is that? </p>&#xA;&#xA;<p>If instead of trying to call a function on the parent, I had those components respond with a function in the CheckBoxWithLabel component, then they work fine. It's only when I set a function in the parent that the checkboxes stop working and the toggleCheckBox function's not called.</p>&#xA;&#xA;<pre><code> toggleCheckBox: function(){&#xA;   console.log('toggle');&#xA; }&#xA; render: function(){&#xA;        return (  &#xA;&#xA;          &lt;div&gt;&#xA;          &lt;CheckboxWithLabel labelOn="On" labelOff="Off" item='blah' onCheck={this.toggleCheckBox} /&gt;&#xA;          &lt;ul&gt;&#xA;            {&#xA;            this.props.formElements.map(function(item) {&#xA;&#xA;            return &lt;CheckboxWithLabel labelOn="On" labelOff="Off" item={item} onCheck={this.toggleCheckBox} /&gt;&#xA;&#xA;            })&#xA;            }               &#xA;        &lt;/ul&gt;&#xA;          &lt;/div&gt;&#xA;          );&#xA;        }&#xA;</code></pre>&#xA;
<p>The function inside of map needs to be bound so that it can use the correct instance of <code>this</code> to retrieve the function: </p>&#xA;&#xA;<pre><code>render: function(){&#xA;  return (  &#xA;    &lt;div&gt;&#xA;      &lt;CheckboxWithLabel labelOn="On" labelOff="Off" item='blah' onCheck={this.toggleCheckBox} /&gt;&#xA;      &lt;ul&gt;&#xA;      {&#xA;        this.props.formElements.map(function(item) {&#xA;          return &lt;CheckboxWithLabel labelOn="On" labelOff="Off" item={item} onCheck={this.toggleCheckBox} /&gt;&#xA;        }.bind(this));&#xA;      }               &#xA;      &lt;/ul&gt;&#xA;    &lt;/div&gt;&#xA;  );&#xA;}&#xA;</code></pre>&#xA;
