How to make an easily embeddable React component that displays any other component's state when used as its child?
<p>I'd like to make a React component that, when inserted inside any other component, will display its state in a DIV floated to the right of the screen. Let name it <code>StateExplorer</code>.</p>&#xA;&#xA;<p>Because many <code>StateExplorers</code> can be used on the same page, the DIVs will stack with the same <code>z-index</code>.</p>&#xA;&#xA;<p>The challenge here is to make <code>StateExplorer</code> easily non-verbosely embeddable, like this:</p>&#xA;&#xA;<pre><code>&lt;SomeComponent&gt;&#xA;    &lt;StateExplorer/&gt;&#xA;    ....&#xA;&lt;/SomeComponent&gt;&#xA;</code></pre>&#xA;&#xA;<p>The particular issue here is: how do I hook on componentWillUpdate() so that the StateExplorer always displays up-to-date state? I could use a mixin, but that throws 2 problems:</p>&#xA;&#xA;<ol>&#xA;<li><code>componentWillUpdate()</code> can be implemented already in the parent component</li>&#xA;<li>adding a mixing adds more verbosity; the ideal case is to just add <code>StateExplorer</code> and nothing more.</li>&#xA;</ol>&#xA;&#xA;<p>P.S. I know about React Debug Tools, but it's not so convenient in some cases and adds extra steps before you can see state of a single component.</p>&#xA;
<p>Sounds like what you need is a <strong>Higher-order component</strong>. &#xA;Here is a good article about it: <a href="https://medium.com/@dan_abramov/mixins-are-dead-long-live-higher-order-components-94a0d2f9e750" rel="nofollow">https://medium.com/@dan_abramov/mixins-are-dead-long-live-higher-order-components-94a0d2f9e750</a></p>&#xA;
