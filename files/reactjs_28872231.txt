Render reactjs component in multiple div tags
<p>I want to create a React.JS component which can be rendered in multiple places using the <em>classname</em> of <code>div</code> elements instead of an <em>id</em>.</p>&#xA;&#xA;<p>The regular way to render a component is this:</p>&#xA;&#xA;<pre><code>React.render(&lt;Component/&gt;, document.getElementById('id'))&#xA;</code></pre>&#xA;&#xA;<p>while I wonder if it' possible to do this:</p>&#xA;&#xA;<pre><code>React.render(&lt;Component/&gt;, document.getElementsByClassName)?&#xA;</code></pre>&#xA;
<p><code>React.render</code> takes a single DOM element. If you want to render it more than once, simply iterate over the nodes.</p>&#xA;&#xA;<pre><code>function renderToElements(toRender, elements) {&#xA;  for (var i = 0; i &lt; elements.length; i++) {&#xA;    React.render(toRender, elements[i]);&#xA;  }&#xA;}&#xA;&#xA;renderToElements(..., document.getElementsByClassName("className"));&#xA;</code></pre>&#xA;
<p>Just adding onto @Michelle Tilley's answer, I really wanted to do this technique while grabbing a dataset from each element to pass in as a prop. The key was to pass the <strong>ComponentName</strong> without quotes or brackets and then to createElement with it :</p>&#xA;&#xA;<pre><code>let elems = document.getElementsByClassName("class_name");&#xA;&#xA;function renderToElements(toRender, elements, dataset) {&#xA;  for (var i = 0; i &lt; elements.length; i++) {&#xA;    let passId = elements[i].dataset[dataset];&#xA;    let renderEl = React.createElement(toRender, {id: passId})&#xA;    ReactDOM.render(renderEl, elements[i]);&#xA;  }&#xA;}&#xA;&#xA;renderToElements(ComponentName, elems, 'id')&#xA;</code></pre>&#xA;
