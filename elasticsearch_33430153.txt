How to dynamically update the synonym filter data base
<p>Is it possible to dynamically update my synonym filter data base in Elasticsearch. As Im doing a synonym training process with my app, is it possible to add new words to the synonym words data base dynamically in elasticsearch?</p>&#xA;
<p>If your synonyms are kept in a file, then you need to update the file, close the index and then re-open it and it should pick up the new synonyms automatically.</p>&#xA;
<pre><code>curl -XPOST 'localhost:9200/my-index/_close'&#xA;echo&#xA;echo updating new Synynom database&#xA;curl -XPUT 'localhost:9200/my-index/_settings' -d @analyzer.json&#xA;echo&#xA;echo Opening index&#xA;curl -XPOST 'localhost:9200/my-index/_open'&#xA;</code></pre>&#xA;&#xA;<p>analyzer.json file looks like below - </p>&#xA;&#xA;<pre><code>{&#xA;  "analysis": {&#xA;    "analyzer": {&#xA;      "his-synonym": {&#xA;        "tokenizer": "whitespace",&#xA;        "filter": [&#xA;          "lowercase",&#xA;          "my-synonym"&#xA;        ]&#xA;      }&#xA;    },&#xA;    "filter": {&#xA;      "my-synonym": {&#xA;        "type": "synonym",&#xA;        "synonyms": [&#xA;          "big,large",&#xA;          "run,jogg"&#xA;        ]&#xA;      }&#xA;    }&#xA;  }&#xA;}&#xA;</code></pre>&#xA;
