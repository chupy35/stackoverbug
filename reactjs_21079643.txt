How to model a HTML table with two cells per column in React?
<p>I've been learning React recently, and I'm running into a problem trying to model and render a HTML table of the following structure (note the columns which are split into two using the colspan attribute for the headers).</p>&#xA;&#xA;<pre><code>+---------------------+---------------------+-----+-------------------+&#xA;|     Col 1 Header    |     Col 2 Header    | ... | Col N header      |&#xA;+----------+----------+----------+----------+-----+---------+---------+&#xA;| Data 1A  | Data 1B  | Data 2A  | Data 2B  | ... | Data NA | Data NB |&#xA;+----------+----------+----------+----------+-----+---------+---------+&#xA;| Data 1A  | Data 1B  | Data 2A  | Data 2B  | ... | Data NA | Data NB |&#xA;+----------+----------+----------+----------+-----+---------+---------+&#xA;</code></pre>&#xA;&#xA;<p>I've modelled the table like this:</p>&#xA;&#xA;<p>Main component <code>Table</code>, which renders:</p>&#xA;&#xA;<pre><code>&lt;table&gt; &#xA;  &lt;thead&gt;&#xA;    &lt;tr&gt;&#xA;    {headers}  &lt;!-- &lt;Header/&gt; components --&gt;&#xA;    &lt;/tr&gt;&#xA;  &lt;/thead&gt;&#xA;  &lt;tbody&gt;&#xA;    {rows}     &lt;!-- &lt;Row/&gt; components --&gt;&#xA;  &lt;/tbody&gt; &#xA;&lt;/table&gt;&#xA;</code></pre>&#xA;&#xA;<p>The <code>Header</code> components render as <code>&lt;th colSpan="2"&gt;Col 1 Header&lt;/th&gt;</code></p>&#xA;&#xA;<p>Each <code>Row</code> renders as</p>&#xA;&#xA;<pre><code>&lt;tr&gt;&#xA;  {cells}    &lt;!-- &lt;Cell/&gt; components --&gt;&#xA;&lt;/tr&gt;&#xA;</code></pre>&#xA;&#xA;<p>Now the <code>Cell</code> component is where I run into trouble as I want to split each column into two sub columns (as the header columns have <code>colSpan="2"</code>.)</p>&#xA;&#xA;<p>Since <a href="http://facebook.github.io/react/docs/component-specs.html#render">ReactComponent's render()</a> method has to return a single child component, I can't figure out how to return the two cells like this: <code>&lt;td&gt;Data 1A&lt;/td&gt;&lt;td&gt;Data 1B&lt;/td&gt;</code></p>&#xA;&#xA;<p>In non-table situations I could return something like </p>&#xA;&#xA;<pre><code>&lt;div&gt;&#xA;   &lt;div class="subcol1"&gt;Data 1A&lt;/div&gt;&#xA;   &lt;div class="subcol2"&gt;Data 1B&lt;/div&gt;&#xA;&lt;/div&gt;&#xA;</code></pre>&#xA;&#xA;<p>but I can't seem to figure out how to achive this with a table. Maybe the way I've designed the component structure is a bit off?</p>&#xA;
<p>Right now it's impossible to make a component in React that renders as two table cells. In the future it may be possible but for now, your best bet is probably to make a plain JS function that returns the cells:</p>&#xA;&#xA;<pre><code>function cellsForData(data) {&#xA;    return [&#xA;        &lt;td&gt;Data 1A&lt;/td&gt;,&#xA;        &lt;td&gt;Data 1B&lt;/td&gt;&#xA;    ];&#xA;}&#xA;</code></pre>&#xA;&#xA;<p>You can include this in your <code>cells</code> array for your table component and everything should work.</p>&#xA;
