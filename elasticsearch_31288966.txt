How to retrieve all document ids (_id) in a specific index
<p>I am trying to retrieve all documents in an index, while getting only the _id field back.</p>&#xA;&#xA;<p>Basically I want to retrieve all the document ids I have.</p>&#xA;&#xA;<p>While using:</p>&#xA;&#xA;<pre><code>{&#xA;   "query": {&#xA;       "match_all": {}&#xA;   },&#xA;   "fields": []&#xA;}&#xA;</code></pre>&#xA;&#xA;<p>The hits I get contain: "_index", "_type", "_id" , "_score", "_source" &#xA;which is way more then I need.</p>&#xA;&#xA;<p><strong>Edit(Answer)</strong>:&#xA;So my problem was that I used KOPF to run the queries, and the results were not accurate (got the _source and some more..)! When using curl I got the correct results!</p>&#xA;&#xA;<p>So the above query actually achieved what I needed!&#xA;You can also use:</p>&#xA;&#xA;<pre><code>{&#xA;   "query": {&#xA;      "match_all": {}&#xA;   },&#xA;   "_source": false,&#xA;}&#xA;</code></pre>&#xA;&#xA;<p>or</p>&#xA;&#xA;<pre><code>{&#xA;   "query": {&#xA;      "match_all": {}&#xA;   },&#xA;   "fields": ["_id"]&#xA;}&#xA;</code></pre>&#xA;
<p>For <strong><em>elasticsearch</em></strong>, only can specific <code>_source</code> fields by using <code>fields</code> array. </p>&#xA;&#xA;<p><code>_index</code>, <code>_type</code>, <code>_id</code>, <code>_score</code> must will be returned by <strong><em>elasticsearch</em></strong>. </p>&#xA;&#xA;<p>there is no way to remove them from response.</p>&#xA;
<p>I am assuming your _id is of your document in index not of index itself.</p>&#xA;&#xA;<p>In new version of elastic search, "_source" is used for retrieving selected fields of your es document because _source fields contains everything you need in elastic search for a es record.</p>&#xA;&#xA;<p>Example: </p>&#xA;&#xA;<p>Let's say index name is "movies" and type is "movie" and you want to retrieve the movieName and movieTitle of all elastic search records.</p>&#xA;&#xA;<pre><code>curl -XGET 'http://localhost:9200/movies/movie/_search?pretty=true' -d '&#xA;{ &#xA;    "query" : { &#xA;        "match_all" : {} &#xA;    },&#xA;    "_source": ["movieName","movieTitle"]&#xA;}'&#xA;</code></pre>&#xA;&#xA;<p>OR <a href="http://localhost:9200/movies/movie/_search?pretty=true&amp;_source=movieName,movieTitle" rel="nofollow noreferrer">http://localhost:9200/movies/movie/_search?pretty=true&amp;_source=movieName,movieTitle</a></p>&#xA;&#xA;<p>By default it return 10 results. For getting n number of records then put &amp;size=n in url</p>&#xA;
