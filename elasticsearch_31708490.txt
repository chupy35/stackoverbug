Elasticsearch match all except specified fields query, exclude
<p>My indexed Elasticsearch documents include many fields. I've been using <a href="https://www.elastic.co/guide/en/elasticsearch/reference/1.6/query-dsl-match-all-query.html" rel="nofollow">match_all</a> query to get results. There are a few fields I'd like to exclude from match_all, is this possible?</p>&#xA;
<p>In Elasticsearch you can use <a href="https://www.elastic.co/guide/en/elasticsearch/reference/current/search-request-fields.html#partial" rel="nofollow">partial</a> fields to filter fields. </p>&#xA;&#xA;<p>Example:</p>&#xA;&#xA;<pre><code>{&#xA;    "query": {&#xA;        "match_all": {}&#xA;    },&#xA;    "partial_fields": {&#xA;        "partial1": {&#xA;            "exclude": ["excludeField1", "excludeField2"]&#xA;        }&#xA;    }&#xA;}&#xA;</code></pre>&#xA;
<p>With ElasticSearch 2.x, you can use <a href="https://www.elastic.co/guide/en/elasticsearch/reference/2.3/search-request-source-filtering.html" rel="nofollow">source filtering</a>, example:</p>&#xA;&#xA;<pre><code>{&#xA;    "_source": {&#xA;        "include": [ "obj1.*", "obj2.*" ],&#xA;        "exclude": [ "*.description" ]&#xA;    },&#xA;    "query" : {&#xA;        "term" : { "user" : "kimchy" }&#xA;    }&#xA;}&#xA;</code></pre>&#xA;&#xA;<p>For a GET request you may also pass it <a href="https://www.elastic.co/guide/en/elasticsearch/reference/2.3/docs-get.html#get-source-filtering" rel="nofollow">via the url</a>.</p>&#xA;
<p>For this question I will give you the example</p>&#xA;&#xA;<pre><code>GET index_name/index_type/_search&#xA;{&#xA;  "_source": {&#xA;&#xA;  "exclude": [&#xA;    ]&#xA;  },&#xA;  "query": {&#xA;    "match_all": {}&#xA;  }&#xA;}&#xA;</code></pre>&#xA;
