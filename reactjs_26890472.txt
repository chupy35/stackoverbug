How to count the number of Grandchildren
<p>New to React and JSX so apologies if this is a very basic question.</p>&#xA;&#xA;<p>I have a "GrandParent" node and I can count the number of "Parent" nodes using:</p>&#xA;&#xA;<pre><code> {React.Children.count(this.props.children)&#xA;</code></pre>&#xA;&#xA;<p>However - given a particular "Parent" - how can I return the count of their children?&#xA;I can do it at the parent level, but can't figure out how to do it at the GrandParent level.</p>&#xA;&#xA;<p>As usual - probably best to explain by example, so I have an example at:&#xA;<a href="http://jsfiddle.net/zuoza31g/1/" rel="nofollow">http://jsfiddle.net/zuoza31g/1/</a></p>&#xA;&#xA;<p>I would like my output to be:</p>&#xA;&#xA;<pre><code>I am the grandfather 'Grandfather'&#xA;I have '1' children&#xA;I have '4' grandchildren&#xA;&#xA;I am the father 'Father' &#xA;I have '4' children&#xA;&#xA;I am a son 'Son 1'&#xA;I am a son 'Son 2'&#xA;I am a son 'Son 3'&#xA;I am a son 'Son 4'&#xA;</code></pre>&#xA;&#xA;<p>It's part of a much bigger problem, so I'm simplified it down to this example .. </p>&#xA;&#xA;<p>Can anyone help please?</p>&#xA;
