Elasticsearch with HTTPS POST/PUT/GET request
<p>Is there any document available to setup Elasticsearch to serve connection over SSL ?. And is the option available on ELS ?.</p>&#xA;&#xA;<p>Ex :  curl -XGET <a href="https://localhost:9200/" rel="nofollow">https://localhost:9200/</a></p>&#xA;&#xA;<p>Thanks.</p>&#xA;
<p>You can setup your code to call SSL. What language are you using? Are you using a custom client or one provided by ElasticSearch (<a href="https://www.elastic.co/guide/index.html" rel="nofollow">Find a list of clients supplied by ElasticSearch here</a>)? If you are using a client you can configure them to use SSL (<a href="https://www.elastic.co/guide/en/elasticsearch/client/php-api/current/_configuration.html" rel="nofollow">Here is an example for PHP's client</a>).</p>&#xA;&#xA;<p>I use SSL by using PHP's curl library. I do not use an clients, but instead a custom client. The following will list the stats of ElasticSearch from an SSL connection.</p>&#xA;&#xA;<pre><code>&lt;?php&#xA;$ch = curl_init('https://localhost:9200/_stats');&#xA;curl_exec($ch); // this outputs to the browser. &#xA;// You can capture it using PHP's output buffer &#xA;// http://php.net/manual/en/function.ob-start.php&#xA;&#xA;curl_close($ch);&#xA;</code></pre>&#xA;
