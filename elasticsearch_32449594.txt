Reporting Tools like Elastic Search with data-level Authorization
<p>Looking for a reporting tool / feature / open source which can support data-level authorization. I think Elastic Search does not provide data level authorization rather it is based only on cluster / indices level.</p>&#xA;
<p>With the release of <a href="https://www.elastic.co/blog/shield-and-watcher-2-0-0-beta2-released" rel="nofollow noreferrer">Shield 2</a> (still in beta though) you can now set up <a href="https://www.elastic.co/guide/en/shield/current/setting-up-field-and-document-level-security.html" rel="nofollow noreferrer">field- and document-level security</a></p>&#xA;&#xA;<pre><code>roleX:&#xA;  indices:&#xA;    '*':&#xA;      privileges: read&#xA;      fields:&#xA;        - fieldA             &lt;-- only users with roleX can see fieldA&#xA;        - fieldB&#xA;&#xA;roleY:&#xA;  indices:&#xA;    '*':&#xA;      privileges: read&#xA;      fields:&#xA;        - fieldB&#xA;</code></pre>&#xA;
