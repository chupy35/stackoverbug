Removing an index from an alias in Elasticsearch
<p>How can we remove an index from an alias? That is, I no longer need to associate an index to an alias.</p>&#xA;&#xA;<p>My second question is - can an alias exist without any indexes?</p>&#xA;
<p>Add an alias:</p>&#xA;&#xA;<pre><code>POST _aliases&#xA;{&#xA;  "actions": [&#xA;    {&#xA;      "add": {&#xA;        "index": ".marvel-2015.06.05",&#xA;        "alias": "alias1"&#xA;      }&#xA;    },&#xA;    {&#xA;      "add": {&#xA;        "index": ".marvel-2015.06.04",&#xA;        "alias": "alias1"&#xA;      }&#xA;    }&#xA;  ]&#xA;}&#xA;</code></pre>&#xA;&#xA;<p>List the alias:</p>&#xA;&#xA;<pre><code>GET /alias1/_alias&#xA;</code></pre>&#xA;&#xA;<p>Remove one index:</p>&#xA;&#xA;<pre><code>POST _aliases&#xA;{&#xA;  "actions": [&#xA;    {&#xA;      "remove": {&#xA;        "index": ".marvel-2015.06.05",&#xA;        "alias": "alias1"&#xA;      }&#xA;    }&#xA;  ]&#xA;}&#xA;</code></pre>&#xA;&#xA;<p>And "no", an alias cannot exists without indices: <a href="https://github.com/elastic/elasticsearch/issues/7864" rel="noreferrer">https://github.com/elastic/elasticsearch/issues/7864</a></p>&#xA;
<p>In case you don't know the name of index and you want to remove alias from all indexes</p>&#xA;&#xA;<pre><code>Post: /_aliases&#xA;{&#xA; "actions" : [&#xA;    { "remove" : { "index" : "*", "alias" : "nameOfAlias" } }&#xA; ]&#xA;}&#xA;</code></pre>&#xA;
