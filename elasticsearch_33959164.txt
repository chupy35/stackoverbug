Search a text in multiple fields
<p>In Elastic-search, i know how to query for multiple fields with different text match from the <a href="https://www.elastic.co/guide/en/elasticsearch/guide/current/multi-query-strings.html#multi-query-strings" rel="nofollow">link</a> . but is there a optimal way to search a text in multiple fields? thanks</p>&#xA;
<p>Indeed there's a query for this purpose, it's called <code>multi_match query</code>. </p>&#xA;&#xA;<pre><code>GET my_index/my_type/_search&#xA;{&#xA;  "query": {&#xA;    "multi_match": {&#xA;      "query": "text to search",&#xA;      "fields": ["my_field1", "my_field2"]&#xA;    }&#xA;  }&#xA;}&#xA;</code></pre>&#xA;&#xA;<p>What this query will do is a search of <code>text to search</code> on both fields <code>my_field1</code> and <code>my_field2</code>.</p>&#xA;&#xA;<p>Here is the documentation about the <a href="https://www.elastic.co/guide/en/elasticsearch/reference/current/query-dsl-multi-match-query.html?q=multi%20ma" rel="nofollow">Multi Match Query</a></p>&#xA;
