Installing bigdesk in ElasticSearch
<p>I'm trying to install bigdesk in ElasticSearch, I downloaded the plugin and copied it uncompressed in the plugin directory</p>&#xA;&#xA;<p>When I try to access to <a href="http://localhost:9200/_plugin/bigdesk" rel="nofollow">http://localhost:9200/_plugin/bigdesk</a> I just see a white page.&#xA;I installed Marvel as well and I didn't have any problem.&#xA;I read the documentation in ES and it doesn't say to do anything else to install a new plugin.</p>&#xA;
<p>You might need to enable JSONP in your <code>elasticsearch.yml</code> configuration file.</p>&#xA;&#xA;<pre><code>http.jsonp.enable: true&#xA;</code></pre>&#xA;&#xA;<p>Also make sure to append a slash at the end of the URL</p>&#xA;&#xA;<pre><code>http://localhost:9200/_plugin/bigdesk/&#xA;                                     ^&#xA;                                     |&#xA;                              add slash here&#xA;</code></pre>&#xA;&#xA;<p>I've caught myself wondering, too, a few times because of this.</p>&#xA;&#xA;<p><strong>UPDATE</strong></p>&#xA;&#xA;<p>Since you've installed the plugin manually, make sure to unzip the package in the <code>/plugins/bigdesk/_site</code> folder since this is a site plugin.</p>&#xA;
