How to show the state of a checkbox with used 'linkState' in reactjs?
<p>I want to use <code>linkState</code> for a checkbox, but I found I can't get the state back and show it:</p>&#xA;&#xA;<pre><code>var Item = React.createClass({&#xA;  mixins: [React.addons.LinkedStateMixin],&#xA;  getInitialState: function() {&#xA;    return {&#xA;      ok: true&#xA;    }&#xA;  },&#xA;  render: function() {&#xA;     return (&#xA;     &lt;div&gt;&#xA;       &lt;input type="checkbox" checkedLink={this.linkState('ok')} /&gt;&#xA;       Is it OK: [{this.state.ok}]&#xA;       &lt;/div&gt;&#xA;     )&#xA;  }&#xA;});&#xA;React.render(&lt;Item /&gt;, document.body);&#xA;</code></pre>&#xA;&#xA;<p>The key line is: </p>&#xA;&#xA;<pre><code>Is it OK: [{this.state.ok}]&#xA;</code></pre>&#xA;&#xA;<p>But the state is not shown. Is it correct to use <code>this.state.ok</code> to show it?</p>&#xA;&#xA;<p>Live demo: <a href="http://jsbin.com/detese/2/edit" rel="nofollow">http://jsbin.com/detese/2/edit</a></p>&#xA;
<p>Your main code is correct, the actual problem is with how you're debugging the code.  </p>&#xA;&#xA;<ul>&#xA;<li><code>&lt;div&gt;{true}&lt;/div&gt;</code> renders as <code>&lt;div&gt;true&lt;/div&gt;</code>  </li>&#xA;<li><code>&lt;div&gt;{false}&lt;/div&gt;</code> renders as <code>&lt;div&gt;&lt;/div&gt;</code></li>&#xA;</ul>&#xA;&#xA;<p>To just drop state into a component's render I use this snippet:</p>&#xA;&#xA;<pre><code>&lt;pre&gt;{JSON.stringify(this.state, null, 4)}&lt;/pre&gt;&#xA;</code></pre>&#xA;&#xA;<p><a href="http://jsbin.com/fazapisocu/2/edit?html,js,output" rel="nofollow">jsbin</a></p>&#xA;&#xA;<p>If you have a boolean and want to render the string representation, do <code>String(thatBoolean)</code>.  React will always render your string exactly as you tell it to, but there are quite a few edge cases for non-string values.</p>&#xA;
