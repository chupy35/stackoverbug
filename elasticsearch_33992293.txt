<p>I'm new to elastic search. How to generate elastic search equivalent query for &#xA;select * from response where <code>pnrno='sampleid'</code></p>&#xA;&#xA;<p>I know we have to use 'filter' option in elastic search.but we do not need any ranking. <p>you are correct , you can use filtered query with query clause empty and filters.Filtering a set of documents is to filter the sets upon which query acts to furthur filter/match and calculate relevance.Filters are like bool either match or reject(1/0).</p>&#xA;&#xA;<pre><code>{&#xA;    "query": {&#xA;        "filtered": {&#xA;            "filter": {&#xA;                "bool": {&#xA;                    "must": [{&#xA;                        "term": {&#xA;                            "FIELD": "VALUE"&#xA;                        }&#xA;                    }]&#xA;                }&#xA;            }&#xA;        }&#xA;    }&#xA;}&#xA;</code></pre>&#xA;
<p>The usual way of achieving this is by using the <a href="https://www.elastic.co/guide/en/elasticsearch/reference/current/query-dsl-constant-score-query.html" rel="nofollow"><code>constant_score</code> query</a> with an embedded <a href="https://www.elastic.co/guide/en/elasticsearch/reference/current/query-dsl-term-query.html" rel="nofollow"><code>term</code> filter</a>, like this:</p>&#xA;&#xA;<pre><code>{&#xA;  "query": {&#xA;    "constant_score": {&#xA;      "filter": {&#xA;        "term": {&#xA;          "pnrno": "sampleid"&#xA;        }&#xA;      }&#xA;    }&#xA;  }&#xA;}&#xA;</code></pre>&#xA;
