Elasticsearch increase term limit
<p>I have this as part of a bool query</p>&#xA;&#xA;<pre><code>      "should": [&#xA;        {&#xA;          "term": {&#xA;            "column1.raw": "value1"&#xA;          }&#xA;        },&#xA;        {&#xA;          "term": {&#xA;            "column1.raw": "value2"&#xA;          }&#xA;        }&#xA;      ]&#xA;</code></pre>&#xA;&#xA;<p>Here the term section will repeat multiple times as per the value of an array which I am using to calculate it. If I limit this array to 500 it is working. But it is not working if the array count is greater than 1000. How can I increase this limit ? most of the times, I have above 4000 values in the array and so the term filter need to repeat for 4000 times.
