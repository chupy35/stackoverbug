How to move data from one Elasticsearch index to another using the Bulk API
<p>I am new to Elasticsearch. How to move data from one Elasticsearch index to another using the Bulk API?</p>&#xA;
<p>I'd suggest using <a href="https://www.elastic.co/guide/en/logstash/current/index.html" rel="nofollow">Logstash</a> for this, i.e. you use one <a href="https://www.elastic.co/guide/en/logstash/current/plugins-inputs-elasticsearch.html" rel="nofollow"><code>elasticsearch</code> input</a> plugin to retrieve the data from your index and another <a href="https://www.elastic.co/guide/en/logstash/current/plugins-outputs-elasticsearch.html" rel="nofollow"><code>elasticsearch</code> output</a> plugin to push the data to your other index. </p>&#xA;&#xA;<p>The config logstash config file would look like this:</p>&#xA;&#xA;<pre><code>input {&#xA;  elasticsearch {&#xA;   hosts =&gt; "localhost:9200"&#xA;   index =&gt; "source_index"           &lt;--- the name of your source index&#xA;  }&#xA;}&#xA;filter {&#xA; mutate {&#xA;  remove_field =&gt; [ "@version", "@timestamp" ]&#xA; }&#xA;}&#xA;output {&#xA; elasticsearch {&#xA;   host =&gt; "localhost"&#xA;   port =&gt; 9200&#xA;   protocol =&gt; "http"&#xA;   manage_template =&gt; false&#xA;   index =&gt; "target_index"           &lt;---- the name of your target index&#xA;   document_type =&gt; "your_doc_type"  &lt;---- make sure to set the appropriate type&#xA;   document_id =&gt; "%{id}"&#xA;   workers =&gt; 5&#xA; }&#xA;}&#xA;</code></pre>&#xA;&#xA;<p>After <a href="https://www.elastic.co/guide/en/logstash/current/getting-started-with-logstash.html" rel="nofollow">installing Logstash</a>, you can run it like this:</p>&#xA;&#xA;<pre><code>bin/logstash -f logstash.conf&#xA;</code></pre>&#xA;
