How to add documents to existing index in elasticsearch
<p>Am using Elasticsearch 1.4. My requirement is I will have data every hour and that needs to be uploaded. So the approach that I have taken is to create an index - "demo" and upload the data. So, the first hour data gets inserted.  Now, my question is how to append the subsequent hours data into this index.</p>&#xA;&#xA;<pre><code>PUT /demo/userdetails/1&#xA;{&#xA;    "user" : "kimchy",&#xA;    "message" : "trying out Elastic Search"&#xA;}&#xA;</code></pre>&#xA;&#xA;<p>Now I am trying to add another document</p>&#xA;&#xA;<pre><code>{"user": "swarna","message":"hi"}&#xA;</code></pre>&#xA;
<p>You simply need to PUT the additional documents. In your example above you did</p>&#xA;&#xA;<pre><code>PUT /demo/userdetails/1 { "user" : "kimchy", "message" : "trying out Elastic Search" }&#xA;</code></pre>&#xA;&#xA;<p>Now you would do this:</p>&#xA;&#xA;<pre><code>PUT /demo/userdetails/2 {"user": "swarna","message":"hi"}&#xA;</code></pre>&#xA;&#xA;<p>In you command there demo is the index, userdetails is the type, and the number is the document id.  If you omit the document id ES will make one up for you.</p>&#xA;
