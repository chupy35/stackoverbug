<p>I got the issue. </p>&#xA;&#xA;<p>I had updated config/elasticsearch.yml  for cluster.name and node.name like below </p>&#xA;&#xA;<pre><code>cluster.name:my_elastic_cluster  &#xA;node.name:"EL1_NODE"&#xA;</code></pre>&#xA;&#xA;<blockquote>&#xA;  <p>But look like parser doesn't like this format. Its need spaces with&#xA;  <strong>":"</strong> delimiter</p>&#xA;</blockquote>&#xA;&#xA;<p>I added one space before and after <strong>:</strong> and it worked !!! </p>&#xA;&#xA;<pre><code>&gt; cluster.name : my_elastic_cluster  &#xA;&gt; node.name : "EL1_NODE"&#xA;</code></pre>&#xA;
<p>Best practice is : Do not put any space before any key. Also define a key value in the yml file is to put a key and then a space, then a colon and then again a space, then value.&#xA;e.g.  cluster.name : my_elastic_cluster</p>&#xA;
