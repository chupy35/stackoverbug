Elastic Wild card search not working with space
<p>I am trying to make search from elastic search index, using wildcard filter, it working properly till i use "space" in my search criteria. </p>&#xA;&#xA;<p>Ex : if value in elastic index is "This is test value"</p>&#xA;&#xA;<p>when i use search value : "This*" --> Working, fetching the value&#xA;when i use search value : "This is*"--> not working, resulting in zero match</p>&#xA;&#xA;<p>my code looks as follows</p>&#xA;&#xA;<p>QueryBuilders.<strong>wildcardQuery</strong>(<br>&#xA;ElasticSearchUtil.FIELDNAME, searchValue.toLowerCase())</p>&#xA;&#xA;<p>how can i handle space and make my search successful?</p>&#xA;&#xA;<p>regards&#xA;UI</p>&#xA;
<p>The wildcard query works only with a single token. What you are trying to do is to find one token followed by another token with the second token being specified by a prefix. This can be achieved by using <a href="http://www.elastic.co/guide/en/elasticsearch/reference/current/query-dsl-match-query.html#_match_phrase_prefix" rel="noreferrer">match phrase prefix</a> query. Your query would look like this:</p>&#xA;&#xA;<p>QueryBuilders.matchPhrasePrefixQuery(ElasticSearchUtil.FIELDNAME, searchValue)</p>&#xA;&#xA;<p>Please note that the <code>searchValue</code> shouldn't have "*" at the end. </p>&#xA;
