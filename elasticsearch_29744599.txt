Bool query or filter query in Elastic Search
<p>I have an expression/query : ("text" started_with "hello" OR "text" started_with "hi") OR ("select" is "first_option" AND "correct" is "true" ) . Can you show me the way to use both of bool and filter query correctly in Elastic Search to solve this expression?</p>&#xA;
<p>You can achieve the given expression/query using "<a href="http://www.elastic.co/guide/en/elasticsearch/reference/0.90/query-dsl-prefix-query.html" rel="nofollow">prefix query</a>" along with <a href="http://www.elastic.co/guide/en/elasticsearch/reference/current/query-dsl-bool-query.html" rel="nofollow">boolean query</a> and <a href="http://www.elastic.co/guide/en/elasticsearch/reference/0.90/query-dsl-term-query.html" rel="nofollow">term query</a>. Try as follows</p>&#xA;&#xA;<pre><code>{&#xA;    "bool": {&#xA;        "should": [&#xA;            {&#xA;                "prefix": {&#xA;                    "text": {&#xA;                        "value": "hello"&#xA;                    }&#xA;                }&#xA;            },&#xA;            {&#xA;                "prefix": {&#xA;                    "text": {&#xA;                        "value": "hi"&#xA;                    }&#xA;                }&#xA;            },&#xA;            {&#xA;                "bool": {&#xA;                    "must": [&#xA;                        {&#xA;                            "term": {&#xA;                                "select": [&#xA;                                    "first_option"&#xA;                                ]&#xA;                            }&#xA;                        },&#xA;                        {&#xA;                            "term": {&#xA;                                "correct": [&#xA;                                    "true"&#xA;                                ]&#xA;                            }&#xA;                        }&#xA;                    ]&#xA;                }&#xA;            }&#xA;        ]&#xA;    }&#xA;}&#xA;</code></pre>&#xA;
